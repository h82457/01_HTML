img 태그

        - 웹 문서에서 사진이나 그림같은 이미지를 삽입하는 용도의 태그
        [속성]
        (속성 : 시작 태그에 작성하는 내용)

        1. src : 삽입할 이미지의 경로를 작성하는 속성
            (파일 위치, 인터넷 주소) 
        2. width / height : 너비 / 높이 지정 속성

        3. alt (대안) : 이미지 설명 작성 속성
                        -> 이미지가 출력되지 않은 경우 화면에 출력됨
                        + 웹 접근성 (스크린 리더가 이미지 설명을 읽어줌)

img 예시

절대 경로로 이미지 출력하기

        절대 경로 : 변하지 않는 기준점으로 부터 목표까지의 경로
        (서울시 중구 남대문로 120 처예 그레이츠 2층)

        - 윈도우 기준 : C드라이브 , D드라이브 이 기준

상대 경로로 이미지 출력하기

        상대 경로 : 현재 파일 / 페이지 위치를 기준으로 목표까지의 경로를 나타내는 것  
        (오른쪽 뒤에 뒤에 ...)
        (학원에서 1층으로 내려가서 오른쪽으로 가면 신한은행)

고정 크기 단위 (px) : 항상 고정된 크기를 지님

가변 크기 단위 (%) : 화면 또는 영역에 대한 상대적인 크기 단위

고정 + 가변 크기

        고정 크기로 설정된 요소 (태그) 내부에서 일정한 크기의 이미지 또는 요소를 출력하고 싶을 때 가변 크기(%)를 이용해서 지정
